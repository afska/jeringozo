# Jeringozo

Spanish-[Jeringoza](https://en.wikipedia.org/wiki/Jeringonza) translator!

# Developing

## Install

```bash
nvm use
npm install
```

## Run the app

```bash
npm start
```

## Build site

```bash
npm run build
```

## Build extension

```bash
./scripts/build-extension.sh
```

## Deploy

```bash
./scripts/deploy.sh <GITHUB_TOKEN>
```
